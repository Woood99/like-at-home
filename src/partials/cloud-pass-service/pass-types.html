Виды пропусков

Разовый

Гость сможет попасть на территорию один раз на одном КПП

Временный

Не ограничивает кол-во посещений, действует в указанный период

Черный список

Система автоматически ограничит доступ для нарушителей

Постоянный

Полный доступ на объект без ограницения кол-ва посещений

Приглашения

Гость сможет сам заполнить данные и приехать в удобное время
